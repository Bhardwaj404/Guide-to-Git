# Guide-to-Git
Steps to add a file to your git repo using vs code:
1.Create a git repo using gihub on your browser
2.open vs code
3.git clone link
4.cd repo name
5.make changes
6.git add .
7.git commit -m "First commit"
8.git push -u main
(also check for branch before commit statement)
(if branch is not main, write git branch -M main)
